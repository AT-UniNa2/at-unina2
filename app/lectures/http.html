ISO OSI model


The Open Systems Interconnection architecture defines a standard way to connect computers

ISO OSI model


Each layer provides a service to the layers above and below:

  • the physical layer handles the transmission of raw bits;
  • the data link layer collects such stream of bits into an aggregate called frame;
  • the network layer handles routing among nodes;
  • the transport layer implements a process-to-process channel;
  • the application layer includes all the protocols of higher level.

World Wide Web


The original goal of the web was to find a way to organize and retrieve informations.

The hypertext, the HTTP protocol and the HTML language were designed to meet that goal.

Gathering resources


When you type www.google.com, your browser

  1. would open a TCP connection to the google server;
  2. retrieves and renders the file called index.html;
  3. generates a new request for each hyperlink present in the page (images or other resources)

Hypertext Transfer Protocol


HTTP is a

  • text-oriented protocol
  • request/response protocol


A standard HTTP message has this general form:

            
START_LINE
MESSAGE_HEADER

MESSAGE_BODY

Request messages


The first line of an HTTP request specifies

  • the operation;
  • the page;
  • the version of HTTP.
            
GET / HTTP/1.1
Host: www.google.it
Please note the blank line at the end of the message

Http request operations


Operation Description
OPTIONS Request information about available options
GET Retrieve documents
HEAD Retrieve metainformations
POST Give information
PUT Store documents
DELETE Delete an url
TRACE Loopback request message
CONNECT Used by proxies

Response messages


The first line of an HTTP response specifies

  • the version of HTTP;
  • a three digit code;
  • a string giving the reason for the response.
            
HTTP/1.1 200 OK
Server:  myServer

some content

Status codes


Code Description
1XX Informational responses
2XX Successful responses
3XX Redirection messages
4XX Client error responses
5XX Server error responses

Most used codes are


200 OK
400 Bad Request
401 Unauthorized
403 Forbidden
404 Not Found
500 Internal Server Error
