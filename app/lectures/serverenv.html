Server Side Development Environment

Goal


We will install all the server side development tools.

It covers:

  • a Java Development Kit;
  • a database server;
  • a Java editor.

Installing JDK and MySQL


This step will be performed on the vagrant machine

            
$ vagrant up
$ vagrant ssh
$ sudo apt-get install default-jdk mysql-server
$ javac -version # This tests the java installation
javac 1.7.0_65
$ mysql --version # This tests the mysql intallation
mysql  Ver 14.14 Distrib 5.5.38, for debian-linux-gnu (i686) using readline 6.3

Installing Intellij


IntelliJ IDEA (Community Edition) is one of the best (imho) open source Java IDE.

Go to http://www.jetbrains.com/idea/download/ and download the last version for your operating system.
